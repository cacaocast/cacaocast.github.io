Épisode 189 - App Store, watchOS, API de réseau, Xcode, Vulkan, DragOnGesture

28 février 2018

Bienvenue dans le cent-quatre-vingt-neuvième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Copies d’écran dans l’App Store - Maintenant jusqu’à 10
  • Ski et surf des neiges - Maintenant sur watchOS
  • Feedback sur les api de réseau - Une demande de Quinn, chez Apple
  • La culture du blâme - Plus dans Xcode/git
  • La couverture de code - Améliorée dans Xcode 9.3
  • Vulkan sur iOS et macOS - Une révolution
  • DragOnGesture - Astuce macOS
