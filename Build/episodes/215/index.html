Épisode 215 - Apps et HTML, MapKit JS, JetBrains Mono, Swift-doc, Plash, astuces

28 janvier 2020

Bienvenue dans le deux-cent-quinzième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Apps et HTML - Un peu plus de restrictions
  • MapKit JS - Pour mettre des cartes sur votre site web
  • JetBrains mono - Police de caractères pour développeurs
  • Swift-doc - Un programme expérimental de génération de documentation pour Swift
  • Plash - Un site web comme fond d’écran
  • Partage de copies d’écran - Saviez-vous qu’il suffit d’appuyer dessus?
  • Messages - Saviez-vous qu’il y avait un champ “Objet”?
