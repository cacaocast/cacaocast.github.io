Épisode 73 - WWDCAlerts.com, UrbanAirship, OCocoa, Langswitch, pngquant, NSLogger, GitLab

04 janvier 2012

Bienvenue dans le soixante-treizième épisode de CocoaCast Cacao! Dans cet épisode,Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • WWDCAlerts.com - Ne ratez pas la WWDC
  • UrbanAirship - Les notifications “push” facilitées
  • Ococoa - On sonne à la porte!
  • Langswitch - Testez les traductions de votre application Mac OS X
  • pngquant - Convertissez vos png 24-bits en 8-bits pour (peut-être) sauver beaucoup d'espace
  • NSLogger - Pour diagnostiquer vos NSLogs à distance
  • GitLab - Gérez vos répertoires git sur votre propre serveur
