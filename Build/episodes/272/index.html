Bienvenue dans le deux-cent-soixante-douzième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Xcode cloud - 25 heures gratuites
  • Chinotto - Gérez tout l’espace utilisé par Xcode
  • Mammoth - Un client Mastodon en code source libre
  • GateEngine - Un moteur de jeu multi-plateforme en Swift
  • TextCase - Conversions de texte
  • HexFiend - Une mise-à-jour de mon éditeur hexadécimal préféré
  • Pow - Maintenant en code source libre
  • MarkWalkthrough - Pour faire des petits tutoriels dans votre application
  • F*FormatStyle - Pour SwiftUI
  • Pas de raccourcis - Comment l'IA ne nous aidera pas à faire autre chose que des gains marginaux pour ce qui est de l'écriture du code
