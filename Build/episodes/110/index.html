Épisode 110 - objc.io, randomuser.me, helpbook, astuce gcc/clang, TWSReleaseNotesView, AFCoreImageSerializer

21 août 2013

Bienvenue dans le cent-dixième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • objc.io - Un nouvel article chaque mois
  • randomuser.me - Pour créer rapidement une banque d’usagers-test
  • helpbook - Un plugin VoodooPad pour créer des fichiers d’aide pour MacOSX
  • Astuce gcc/clang - Saviez-vous que le compilateur retournera une valeur si vous créez une section de code avec des doubles-accolades?
  • TWSReleaseNotesView - Critique pour iOS7
  • AFCoreImageSerializer - Appliquez des filtres lors du téléchargement de vos images
