Épisode 224 - iMac, Security Device Research Program, BackToMac, cedar, Fawkes, Macintosh.js

05 août 2020

Bienvenue dans le deux-cent-vingt-quatrième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • iMac - Mise à jour
  • Security Device Research Program - Pour les experts en sécurité
  • BackToMac - Une conférence en ligne pour les développeurs Mac
  • cedar - Un mini éditeur graphique en ligne de commande
  • Fawkes - Pour limiter les programmes de reconnaissance faciale
  • Macintosh.js - Un Quadra 800 (~1991) en Javascript
