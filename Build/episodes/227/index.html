Épisode 227 - Apple, Github CLI, MetricKit, PinpointKit, SkeletonView, BuildWatch, Construct, Fastlane, Xcode 12, Notes iOS

06 octobre 2020

Bienvenue dans le deux-cent-vingt-septième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Apple - Hi, Speed
  • Github CLI - Interface github en ligne de commande
  • MetricKit - Les outils de diagnostic inclus dans iOS 14
  • PinpointKit - Pour obtenir des bonnes rétro-actions dans vos applications
  • SkeletonView - Pour préparer vos vues avant qu’elles n’apparaissent
  • BuildWatch - Toutes les statistiques Xcode
  • Construct - Une application D&D avec une architecture moderne
  • Fastlane - Tout se fait maintenant avec une clé API
  • Xcode 12 - Facile d’effacer tous les breakpoints désactivés
  • Notes iOS - Astuce pour indenter le texte
